# OpenEdge Insight

Version: 0.1
Status: Draft
Last Updated: 10 July 2026

**Vision statement**

OpenEdge Insight empowers development teams to understand, measure, and modernise their OpenEdge applications through intelligent analysis and actionable insights.

**Mission**

To provide developers, architects, and technical leaders with clear visibility into the health of their OpenEdge applications, helping them make informed decisions, reduce technical debt, and modernise with confidence.

**Problem Statement**

Many OpenEdge applications have evolved over decades.

Development teams often struggle to answer questions such as:

How healthy is our application?
How ready are we for PASOE?
Where is our technical debt?
Which areas present the highest risk?
What should we improve first?
How can we measure progress?

Existing approaches are often manual, time-consuming, and inconsistent.

**Our Solution**

OpenEdge Insight provides development teams with the tools they need to understand, assess, and modernise their OpenEdge applications. By combining intelligent static analysis with actionable recommendations, the platform transforms complex legacy systems into measurable, understandable, and manageable projects.

Core capabilities include:

- Project inventory
- Code quality metrics
- Technical debt analysis
- PASOE readiness assessment
- Dependency analysis
- Migration recommendations
- Actionable health reports


**Guiding Principles**

Every feature should satisfy at least one of these:

- Help developers understand their application.
- Help teams measure progress.
- Reduce migration risk.
- Save developers time.
- Encourage modern development practices.

**Product Vision**

OpenEdge Insight aims to become the trusted intelligence platform for OpenEdge applications by providing meaningful insights that help organisations make better technical decisions throughout the software lifecycle.
  
**Target Audience**

**Primary**

- OpenEdge developers
- Technical Leads
- Software Architects
	
**Secondary**

- Development Managers
- OpenEdge consultants
- Modernisation teams

## Long-Term Vision
OpenEdge Insight will become the trusted intelligence platform for OpenEdge applications, enabling organisations to understand, modernise, and continuously improve their software throughout its lifecycle.

The platform will evolve beyond static code analysis to provide a comprehensive suite of capabilities, including:

* Intelligent project discovery and inventory
* Static code analysis and quality metrics
* Technical debt identification and trend analysis
* Architecture visualisation and dependency mapping
* PASOE and modernisation readiness assessments
* Actionable recommendations and best practice guidance
* Developer tooling through Visual Studio Code integration
* AI-powered assistance through MCP and modern AI platforms
* Test impact analysis and quality validation
* CI/CD integration with automated quality gates
* Enterprise dashboards, reporting, and portfolio-wide insights

OpenEdge Insight will help organisations make informed technical decisions, reduce modernisation risk, and measure progress over time, becoming an essential part of every OpenEdge development team's toolkit.

**Success Measures**

OpenEdge Insight will be successful when:

- Development teams use it to understand the health of their applications.
- Architects use it to support modernisation planning.
- Organisations can measure technical debt over time.
- Developers use it during day-to-day development.
- Teams gain confidence in OpenEdge and PASOE migrations.
- OpenEdge Insight becomes a trusted tool within the OpenEdge community.

  
**Core Principles**

Every feature should help someone make a better technical decision about their OpenEdge application.

OpenEdge Insight is built on four principles:

1. Simple – Easy to install and use.
2. Transparent – Every recommendation should be explainable.
3. Measurable – Progress should be quantifiable.
4. Extensible – The platform should evolve with OpenEdge and integrate with modern developer tools.

**Product Pillars**

OpenEdge Insight will evolve around four strategic pillars:
   

| Pillar      | Purpose                                                  |
| ----------- | -------------------------------------------------------- |
| 🔍 Discover | Understand the application                               |
| 🧠 Analyse  | Identify risks and technical debt                        |
| 📈 Improve  | Recommend modernisation opportunities                    |
| ✅ Validate  | Increase confidence through testing and quality analysis |

OpenEdge Insight is more than a code analysis tool. It is a platform designed to help organisations understand their applications, reduce technical risk, and modernise with confidence.

Every insight should lead to a better decision.

Every recommendation should create value.

Understand. Measure. Modernise.
