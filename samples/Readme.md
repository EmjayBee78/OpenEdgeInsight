# OpenEdge Insight

Understand. Measure. Modernise.

OpenEdge Insight is an engineering intelligence platform for Progress OpenEdge applications.

It helps development teams understand application health, identify technical debt, assess PASOE readiness and modernise with confidence.

## Features

- Project Discovery
- Static Analysis
- Technical Debt
- Dependency Analysis
- Reporting
- VS Code Integration (Planned)
- AI Integration (Planned)
